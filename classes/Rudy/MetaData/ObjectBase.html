build_criteria (more=[], less=[])

Builds a zipped Array from a list of criteria. The list of criteria is made up of metadata object attributes. The list is constructed by taking the adding more, and subtracting less from :rtype, :region, :zone, :environment, :role, :position

Returns [[:rtype, value], [:zone, value], …]
